# Application "Guess Number"

===========================

## About

Web приложение, при запуске которого пользователю предлагается угадать число
от 1 до 100 за 5 попыток (есть возможность выбрать свой диапазон чисел и количество попыток)

## Technology

- HTML5
- SASS/SCSS
- Vanilla JavaScript
- Flexbox
- Gulp
- Node-SASS
